# Privacy Policy URL Configuration

Please host the privacy_policy.html file on a web server and update this URL.

## Required URL:
YOUR_PRIVACY_POLICY_URL_HERE

## Example URLs:
- https://yourcompany.com/privacy/bankvoicealert
- https://yourdomain.github.io/bankvoicealert-privacy
- https://sites.google.com/view/bankvoicealert-privacy

## Free Hosting Options:
1. GitHub Pages (free)
2. Google Sites (free)
3. Netlify (free tier)
4. Vercel (free tier)

## Instructions:
1. Upload privacy_policy.html to your chosen hosting service
2. Copy the public URL
3. Use this URL in Google Play Console when publishing the app
