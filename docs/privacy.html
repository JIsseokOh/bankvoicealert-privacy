띵동 입금알리미 개인정보처리방침

최종 업데이트: 2025년 1월 25일

1. 개인정보 처리 목적

띵동 입금알리미(이하 "앱")는 다음 목적으로 최소한의 권한만을 요청합니다.

권한 목적
알림 접근 권한 은행 앱의 푸시 알림을 감지하여 입금 내역 확인
포그라운드 서비스 백그라운드에서 지속적인 알림 감지 서비스 제공
인터넷 접근 앱 버전 확인 및 업데이트 알림

2. 처리하는 개인정보 항목

가. 자동 수집 정보

  • 은행 앱 푸시 알림 내용 (입금 금액, 입금자명)
  • 입금 일시 및 기록

나. 수집하지 않는 정보

  • 계좌번호, 잔액 등 금융 민감정보
  • 위치정보, 연락처, 통화기록
  • 기기 고유식별자 (IMEI, 광고ID 등)
중요: 모든 데이터는 사용자 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 개인정보 보유 및 이용 기간

구분 보유 기간
입금 기록 데이터 앱 삭제 시까지 (기기 내부 저장)
앱 설정 정보 앱 삭제 시까지 (기기 내부 저장)

앱 삭제 시 모든 데이터가 자동으로 완전히 삭제됩니다.

4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

  • 외부 서버 전송: 없음
  • 광고 네트워크 공유: 없음
  • 분석 서비스 연동: 없음

5. 개인정보 처리 위탁

앱은 개인정보 처리를 외부 업체에 위탁하지 않습니다.

6. 이용자의 권리 및 행사 방법

사용자는 다음과 같은 권리를 행사할 수 있습니다.

  • 권한 철회: 기기 설정에서 앱 권한을 언제든지 해제 가능
  • 데이터 삭제: 앱 삭제를 통해 모든 로컬 데이터 완전 삭제
  • 서비스 중단: 앱 내 설정에서 백그라운드 서비스 비활성화 가능

7. 개인정보 보호를 위한 기술적 조치

  • 모든 데이터는 기기 내부 저장소(Private Storage)에 저장
  • 다른 앱에서 데이터 접근 불가
  • 네트워크 통신 시 HTTPS 암호화 적용

8. 아동의 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만 사용자의 앱 이용을 권장하지 않습니다.

9. 개인정보처리방침 변경

개인정보처리방침이 변경되는 경우, 변경 사항을 앱 업데이트 및 본 페이지를 통해 공지합니다.

  • 공고일: 2025년 1월 25일
  • 시행일: 2025년 1월 25일
