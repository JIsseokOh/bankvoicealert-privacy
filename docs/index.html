은행 입금 알림 개인정보처리방침

최종 업데이트: 2025년 09월 26일

1. 개인정보의 수집 및 이용 목적

은행 입금 알림 앱은 다음과 같은 목적으로 최소한의 권한만을 요청합니다:

  • SMS 수신/읽기 권한: 은행 입출금 문자 메시지를 감지하기 위함
  • 알림 접근 권한: 알림 내용을 읽고 음성으로 변환하기 위함
  • 알림 표시 권한: 포그라운드 서비스 알림 표시 (Android 13 이상)
  • 인터넷 권한: 광고 표시를 위함

2. 수집하는 개인정보 항목

본 앱은 다음과 같은 정보에 접근합니다:

  • SMS 문자 메시지: 은행명, 거래금액, 잔액 정보 추출
  • 알림 내용: 은행 앱 푸시 알림의 입금 정보
  • 기기 정보: 광고 ID (광고 표시 목적)

중요: 모든 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 개인정보의 보유 및 이용 기간

  • 앱 사용 중: 알림 기록은 사용자 기기에 로컬로 저장됩니다
  • 앱 삭제 시: 모든 데이터가 자동으로 삭제됩니다

4. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

4-1. 개인정보의 안전성 확보 조치

본 앱은 개인정보 보호를 위해 다음과 같은 조치를 취합니다:

  • 모든 개인정보는 사용자 기기 내부 저장소에만 보관
  • 외부 서버로의 개인정보 전송 없음
  • 앱 삭제 시 모든 데이터 자동 삭제
  • 루팅된 기기에서의 사용 제한 권고

5. 개인정보의 처리 위탁

본 앱은 개인정보 처리를 외부에 위탁하지 않습니다.

6. 이용자의 권리

사용자는 언제든지:

  • 앱 설정에서 권한을 해제할 수 있습니다
  • 앱을 삭제하여 모든 로컬 데이터를 제거할 수 있습니다
  • 기기 설정에서 광고 맞춤설정을 관리할 수 있습니다

9. 기타

10. 광고 및 분석 서비스

본 앱은 무료 서비스 제공을 위해 Google AdMob 광고를 표시합니다:

  • 광고 식별자(ADID)가 광고 맞춤 설정에 사용될 수 있습니다

  • 사용자는 기기 설정 > Google > 광고에서 광고 맞춤설정을 관리할 수 있습니다
  • 개인정보는 광고 목적으로 별도 수집하지 않습니다
