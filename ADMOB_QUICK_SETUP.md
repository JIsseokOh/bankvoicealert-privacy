# AdMob 실제 광고 빠른 설정 가이드

## 🎯 즉시 실행 단계

### Step 1: AdMob 계정에서 ID 받기

1. [AdMob](https://admob.google.com) 접속
2. **앱 추가** → Android → Play Store URL 입력
3. **광고 단위** 생성:
   - 배너 광고 1개
   - 네이티브 광고 1개
4. 3개의 ID를 받게 됩니다:
   - App ID (예: ca-app-pub-1234567890123456~1234567890)
   - 배너 광고 ID (예: ca-app-pub-1234567890123456/1111111111)
   - 네이티브 광고 ID (예: ca-app-pub-1234567890123456/2222222222)

### Step 2: ID 적용 명령어

받은 ID를 알려주시면 제가 코드에 바로 적용해드리겠습니다.

예시:
```
App ID: ca-app-pub-1234567890123456~1234567890
배너 ID: ca-app-pub-1234567890123456/1111111111
네이티브 ID: ca-app-pub-1234567890123456/2222222222
```

### Step 3: 자동 적용 (제가 해드릴 작업)

1. AndroidManifest.xml 수정
2. AdManager.kt 수정
3. 테스트 디바이스 설정 추가
4. 새 버전 빌드

## ⚠️ 중요 주의사항

### 반드시 지켜야 할 것
1. **자신의 광고 클릭 금지** - 계정 정지 위험
2. **친구/가족에게 광고 클릭 요청 금지**
3. **테스트할 때는 테스트 모드 사용**

### 테스트 방법
1. 앱 실행 후 Logcat 확인
2. "Use RequestConfiguration.Builder().setTestDeviceIds..." 메시지 찾기
3. 해당 ID를 테스트 디바이스로 등록

## 📱 광고 수익 예상

- **배너 광고**: 1000회 노출당 $0.5~$2
- **네이티브 광고**: 1000회 노출당 $2~$10
- 한국 사용자 기준 평균적으로 높은 편

## 🚀 최적화 팁

1. **광고 배치**
   - 메인 화면 하단: 배너 광고
   - 설정 화면: 네이티브 광고

2. **사용자 경험**
   - 광고가 기능을 방해하지 않도록
   - 실수로 클릭하지 않도록 적절한 여백

3. **정책 준수**
   - "광고 클릭하면 기능 해제" 같은 유도 금지
   - 광고임을 명확히 표시

## 준비되셨나요?

AdMob에서 받은 3개의 ID를 알려주시면 즉시 적용하겠습니다!
