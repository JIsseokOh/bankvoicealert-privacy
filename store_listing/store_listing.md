## 앱 설명

### 간단한 설명 (80자 이내)
은행 입출금 문자를 음성으로 읽어주는 편리한 알림 앱

### 자세한 설명 (4000자 이내)
은행 입금 알림은 은행에서 오는 입출금 문자 메시지를 자동으로 감지하여 음성으로 읽어주는 편리한 알림 앱입니다.

**주요 기능:**
• 은행 입출금 문자 자동 감지
• TTS(Text-to-Speech)를 통한 음성 알림
• 다양한 은행 지원
• 사용자 맞춤 설정 가능
• 알림 기록 관리

**이런 분들께 추천합니다:**
• 운전 중 입출금 내역을 확인하고 싶은 분
• 시각이 불편한 분
• 즉시 입출금 알림을 받고 싶은 분
• 핸드폰을 자주 확인하기 어려운 상황에 있는 분

**지원 은행:**
국민은행, 신한은행, 우리은행, 하나은행, 농협은행, 기업은행 등 주요 시중은행

**개인정보 보호:**
• 모든 데이터는 기기에만 저장됩니다
• 외부 서버로 데이터를 전송하지 않습니다
• 안전한 알림 서비스를 제공합니다

## 스크린샷 준비 사항
최소 2개, 최대 8개의 스크린샷 필요:
1. 메인 화면
2. 설정 화면
3. 알림 기록 화면
4. 권한 설정 안내 화면

스크린샷 규격:
- 휴대폰: 최소 320px, 최대 3840px
- 권장: 1080 x 1920px (16:9 비율)
